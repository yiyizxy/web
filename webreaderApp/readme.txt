HTML5 Web App 阅读器:
从设计图分析核心功能：
★页面布局：1 顶部导航：利用 HTML+CSS 及 Base64 格式的图片制作<返回图标 ICON，通过定位技术将顶部导航固定于页面顶部；
2 内容展示区：利用 DIV+CSS 及 CSS3 的新特性对内容区进行排版布局，包括章节标题、章节内容及章节底部的上一章、下一章导航按钮；
3 底部导航：利用 HTML+CSS 及 Base64 格式的图片制作返回目录、字体大小切换、阅读器背景颜色切换、夜间模式切换等 ICON 图标，通过定位技术将底部导航固定于页面底部；
★事件交互：1点击内容展示区中部实现顶部导航和底部导航的显示和隐藏切换，触发滚动条事件时，导航条隐藏；
2 底部导航中点击字体，实现字体大小切换和 Web App 阅读器背景颜色切换；
3 实现页面底部上一章、下一章按钮切换章节信息功能。
★数据交互：1 利用 zepto.min.js 插件进行 DOM 操作；利用 Ajax 技术从服务器端获取数据内容后通过base64 插件对获取的内容解码，通过 escape 将解码后的内容再次进行编码，使内容能被所有计算机读取，最后再进行 decodeURIComponent 解码，最终以文字方式显示在 Web App阅读器的内容展示区中；
2 利用 HTML5 新增的 LocalStorage 技术将 Web App 阅读器上次使用后的章节信息、字体大小、内容展示区背景颜色等信息存储在本地，实现重新进入阅读器后，阅读器具有记忆功能。
